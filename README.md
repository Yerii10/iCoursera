# iCoursera 项目文档

## 项目概述

iCoursera 是一个轻量级的在线课程学习平台，专注于提供简洁直观的课程浏览与视频学习体验。平台支持用户注册、登录、课程列表查看及视频播放等核心功能，整体采用前端技术实现，无需复杂后端环境即可运行，适合作为在线教育类应用的基础演示框架。

## 技术栈

- **核心技术**：HTML5（页面结构）、CSS3（样式定义）、JavaScript（交互逻辑）
- **样式框架**：Tailwind CSS（用于快速构建响应式界面）
- **数据存储**：localStorage（临时存储课程信息，实现页面间数据传递）
- **依赖资源**：通过 CDN 引入 Tailwind CSS，无其他第三方库依赖

## 功能模块说明

### 1. 用户认证模块

#### 注册功能
- **页面文件**：register.html
- **样式文件**：css/register.css
- **脚本文件**：js/register.js
- **核心功能**：
  - 提供姓名、邮箱、密码三个输入字段，支持用户信息填写
  - 表单提交时进行非空验证，确保所有字段均被填写
  - 注册成功后显示提示信息，并自动跳转至登录页面
  - 支持 "已有账号" 快速跳转至登录页
- **界面特点**：
  - 输入框获取焦点时边框变为绿色（#10B981），提升交互反馈
  - 注册按钮采用绿色主题（默认#10B981，hover 时#059669），视觉统一
  - 表单采用卡片式设计，包含阴影效果，增强层次感

#### 登录功能
- **页面文件**：login.html
- **样式文件**：css/login.css
- **脚本文件**：js/login.js
- **核心功能**：
  - 提供邮箱、密码两个输入字段，支持用户账号登录
  - 表单提交时进行非空验证，确保输入完整
  - 登录成功后直接跳转至课程列表页面
  - 支持 "还没有账号" 快速跳转至注册页
- **界面特点**：
  - 输入框获取焦点时边框变为蓝色（#3B82F6），区分于注册页
  - 登录按钮采用蓝色主题（默认#3B82F6，hover 时#2563EB），视觉识别清晰
  - 页面布局与注册页保持一致，保证用户体验连贯性

### 2. 课程列表模块

- **页面文件**：courses.html
- **样式文件**：css/courses.css
- **脚本文件**：js/courses.js
- **核心功能**：
  - 展示 6 门预设课程，每门课程以卡片形式呈现
  - 课程卡片包含封面图、标题、描述及 "查看课程" 按钮
  - 点击 "查看课程" 按钮时，将课程标题和视频地址存入 localStorage，并跳转至视频播放页
  - 支持 "退出登录" 功能，点击后返回登录页面
- **界面特点**：
  - 采用响应式网格布局：移动端 1 列、平板 2 列、桌面端 3 列
  - 课程卡片默认包含轻微阴影，hover 时上浮 5px 且阴影加深（transform: translateY(-5px)），增强交互感
  - 课程封面图固定高度（200px），采用object-fit: cover确保图片比例正确
  - "查看课程" 按钮为蓝色主题，与登录按钮风格统一

### 3. 视频播放模块

- **页面文件**：video.html
- **样式文件**：css/video.css
- **脚本文件**：js/video.js
- **核心功能**：
  - 页面加载时从 localStorage 读取课程标题和视频地址，并渲染到页面
  - 提供标准视频播放器（支持播放 / 暂停、进度条、音量调节等原生控制）
  - 支持 "返回课程" 功能，点击后回到课程列表页面
- **界面特点**：
  - 视频容器采用黑色背景 + 圆角设计，与白色页面背景形成对比
  - 视频播放器宽度自适应容器，高度按比例自动调整
  - 顶部导航栏左侧为 "返回课程" 按钮，中间为页面标题，布局对称

## 页面交互流程

1. **入口流程**：用户首先访问登录页（login.html），可通过页面链接跳转至注册页（register.html）
2. **注册流程**：填写注册信息→提交验证→成功后跳转至登录页
3. **登录流程**：填写登录信息→提交验证→成功后跳转至课程列表页（courses.html）
4. **学习流程**：在课程列表页选择课程→点击 "查看课程"→跳转至视频播放页（video.html）观看课程
5. **返回流程**：视频播放页可通过 "返回课程" 按钮回到课程列表页；课程列表页可通过 "退出登录" 按钮返回登录页

## 使用说明

1. **启动方式**：直接使用浏览器打开login.html文件即可启动项目
2. **注册操作**：
   - 点击登录页的 "立即注册" 链接进入注册页
   - 依次填写姓名、邮箱（格式不限）、密码
   - 点击 "注册" 按钮，提示成功后自动跳转至登录页
3. **登录操作**：
   - 在登录页输入任意邮箱和密码（无需与注册信息匹配，仅做非空验证）
   - 点击 "登录" 按钮进入课程列表页
4. **课程学习**：
   - 在课程列表页点击任意课程卡片的 "查看课程" 按钮
   - 进入视频播放页后自动加载课程视频，可使用播放器控制栏进行播放操作
   - 点击顶部 "返回课程" 按钮可回到课程列表
5. **退出登录**：在课程列表页点击右上角 "退出登录" 按钮，返回登录页

## 界面设计规范

- **色彩方案**：
  - 登录相关元素主色：蓝色（#3B82F6）
  - 注册相关元素主色：绿色（#10B981）
  - 背景色：浅灰（#f7f9fc、#f0f0f0）
  - 文本色：深灰（#555、#333）
- **交互反馈**：
  - 按钮 hover 状态有明确颜色变化（加深色调）
  - 输入框获取焦点时有边框颜色变化
  - 课程卡片 hover 时有上浮和阴影变化效果
- **布局规范**：
  - 所有页面采用容器居中布局（container mx-auto）
  - 表单元素间距统一（space-y-4）
  - 响应式设计适配不同屏幕尺寸
